# 기회레이더 MVP

정부·지자체·공공기관 지원사업 공고를 '신청 여부 판단 카드'로 바꾸는 구독형 정보 서비스의 정적 MVP입니다.

## 현재 포함된 것
- 랜딩페이지
- 무료/유료 가격 구조
- 샘플 리포트
- 브라우저 로컬 저장 기반 수요 검증 폼
- 모바일 반응형 레이아웃

## 배포
폴더 전체를 GitHub 저장소에 올린 뒤 Vercel 또는 Netlify에 연결하면 됩니다.
별도 빌드 명령은 필요 없습니다.

## 실제 이메일 수집 연결
현재 폼은 브라우저 localStorage에만 저장합니다. 공개 배포 전 아래 중 하나와 연결해야 합니다.
- Tally
- Google Forms
- Formspree
- Buttondown/Substack/Beehiiv 구독 폼

## 최초 검증 목표
1. 방문자 100명
2. 이메일 전환율 8% 이상
3. 유료판 관심 클릭 또는 답장 5건 이상
4. 9,900원 선결제 3건 이상이면 정식 제작

## 운영 단위
- 무료: 주 1회, 핵심 공고 3건
- 유료: 주 1회, 10~20건 + 적합도표 + 체크리스트
- 타깃: 소상공인, 예비창업자, 소규모 대행사
